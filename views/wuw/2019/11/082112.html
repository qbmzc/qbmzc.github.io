Manjaro安装

congco 2019-11-08 manjaro

壁纸百度网盘链接 提取码:njv2

# 安装与配置

下载地址manjaro

推荐一个软件rufus刻录镜像.

# 生成可用中国镜像站列表

sudo pacman-mirrors -i -c China -m rank
1

选择速度快的源

# 刷新

sudo pacman -Syyu
1

# 修改pacman.conf

使用方法：在 /etc/pacman.conf 文件末尾添加以下两行： sudo vim /etc/pacman.conf

[archlinuxcn]
Server = https://mirrors.tuna.tsinghua.edu.cn/archlinuxcn/$arch
1
2

社区仓库镜像

安装 archlinuxcn-keyring 包导入GPG key

sudo pacman -Sy archlinuxcn-keyring
1

出现无法锁定database的错误或者等待另一个包管理器退出

sudo rm /var/lib/pacman/db.lck
1

使用yay安装软件
不需要使用root

sudo pacman -S yay
1

sogou-qimpanel

 sogou-qimpanel:error while loading shared libraries: libfcitx-qt.so.0: cannot open shared object file: No such file or directory

 yay -S fcitx-qt4

Gnome On Wayland 用户无法使用 fcitx 由于 wayland 无法读取 ~/.xprofile 中的环境变量，所以请在/etc/environment中加入： 帮助地址

GTK_IM_MODULE=fcitx
QT_IM_MODULE=fcitx
XMODIFIERS=@im=fcitx

# AUR安装

建议手动安装官网版本

# 手动安装， 下载tar.gz包 下载地址

解压

tar -zxvf xxx.tar.gz
1

移动到 /opt目录下

sudo mv xxx /opt/
1

配置jdk环境变量 修改配置文件 /etc/profile

# setting for jdk-oracle
JAVA_HOME=/opt/jdk1.8.0_211
CLASSPATH=.:$JAVA_HOME/lib/tools.jar:$JAVA_HOME/lib/dt.jar
PATH=$JAVA_HOME/bin:$PATH
export JAVA_HOME CLASSPATH PATH
1
2
3
4
5

启用配置

source /etc/profile
1

此时就已经配置完毕了。
