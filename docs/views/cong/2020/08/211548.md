系统已安装`Anaconda3`并且配置了国内的源。

## 创建虚拟环境

```bash
conda --version
    conda 4.8.2
conda create --name tensorflow python=3.7

# 查看环境
conda env list

# 激活环境
conda activate tensorflow
```

## 安装`CUDA&CUDNN`

```bash
yay cuda # 查找对应的包
```


## 安装TensorFlow

```bash
pip install tensorflow-gpu
```

## 验证

```python
import tensorflow as tf
print(tf.__version__)
# 2.3.0
```
