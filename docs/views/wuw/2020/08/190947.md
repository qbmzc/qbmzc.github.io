## 查看CPU信息

```bash
cat /proc/cpuinfo
```

## 内存信息

```bash
cat /proc/meminfo
```

查看内存型号、频率

```bash
sudo dmidecode -t memory
```

## 内核版本

```bash
cat /proc/version
```
