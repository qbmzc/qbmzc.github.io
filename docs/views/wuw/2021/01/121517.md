## 执行sql语句,获取月份

```sql
select id,MONTH(begindate) as month from test
```

## 获取年份

```sql
select id,YEAR(begindate) as year from test
```

## data_format

```sql
SELECT * FROM ashe_project WHERE DATE_FORMAT(create_time,'%Y')=2020;
```
